# trabalho1_Metodos_do_Calculo_Numerico
Nesse trabalho foram fornecidos os arquivos manh.el e manh.xy, que fornecem as
arestas e as coordenadas dos vértices do grafo de ruas da ilha de Manhattan, NY. A
partir desses arquivos foi construída uma matriz Laplaciana L do grafo de ruas, uma
matriz de penalidades P e um vetor b, e foi possível chegar no sistema (L + P)x =
Pb. O objetivo do trabalho é resolver o sistema usando quatro métodos diferentes e
comparar o tempo de solução: Decomposição LU, Jacobi, Gaus-Seidel e Gradientes
Conjugados.
